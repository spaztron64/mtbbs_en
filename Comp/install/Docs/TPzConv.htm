プロパティ

JisByteCodeType: TJisByteCodeType

JISコードの際の1バイトコードのモード設定をします。

btAscii： Asciiコードに設定します 'ESC(B'が付加される （デフォルト値）
btRoman： Romanコードに設定します 'ESC(J'が付加される

　

JisKanaConvType: TJisKanaConvType

JIS,EUC変換の際半角カナの扱い方法

ktNone： 何もしない
ktZenWithDaku： 濁点を含む2バイトコードに変換する（デフォルト値）
例： ﾋﾞﾋﾞﾃﾞ -> ビビデ
ktZenSepaDaku： 濁点を含まず別々にして変換する
例： ﾊﾞﾋﾞﾃﾞ -> ハﾞヒﾞテﾞ

※どちらのプロパティもデフォルト値のままで特に問題ありません

SJisToJis

文字列をShiftJISからJISにコード変換する
function SJisToJis(const s: AnsiString): AnsiString;

sにShiftJIS文字列を指定するとJISコードが戻り値に入る
プロパティJisKanaConvTypeの設定によって半角カタカナの扱いが変化します

　

EucToSJis

文字列をEUCからShiftJISにコード変換する
function EucToSJis(const s: AnsiString): AnsiString;

sにEUC文字列を指定するとShiftJISコードが戻り値に入る
※2bytes固定EUCはサポートしていませんが、現在ほとんど使われていません

　

SJisToEuc

文字列をShiftJISからEUCにコード変換する
function SJisToEuc(const s: AnsiString): AnsiString;

sにShiftJIS文字列を指定するとEUCコードが戻り値に入る
プロパティJisKanaConvTypeの設定によって半角カタカナの扱いが変化します
※2bytes固定EUCはサポートしていませんが、現在ほとんど使われていません

MIMEHeaderEncode

MIMEヘッダ部分をエンコードする (JIS＋Base64）
function MIMEHeaderEncode(const s: AnsiString): AnsiString;

sに文字列を指定すると '=?ISO-2022-JP?B?' の付いたMIMEヘッダにエンコードした符号化文字列が戻り値に入る

　

MIMEHeaderDecode

MIMEヘッダ部分をデコードする (JIS＋Base64）
function MIMEHeaderDecode(const s: AnsiString): AnsiString;

sにMIMEヘッダ文字列を指定するとShiftJISコードが戻り値に入る
複数のブロックでエンコード文字列がある場合もすべて変換できます

　

WebEncode

Ｗｅｂコードにエンコードする（WEBサーバ等で使われる）
function WebEncode(const s: AnsiString): AnsiString;

文字列内で８ビット目が”１”の文字を１６進表記”%nn”に変換する
スペースは(' ')プラス記号('+')に変換されます

　

WebDecode

Ｗｅｂコードにデコードする（WEBサーバ等で使われる）
function WebDecode(const s: AnsiString): AnsiString;

文字列内のエンコードされた%nn部分を通常の文字に変換する
プラス記号('+')はスペース(' ')に変換されます
