以上の１０個のファイルを任意の同一フォルダーにコピーします。

Delphiのメニューから，[コンポーネント（C）｜コンポーネントのインストール（I）...]をクリックします。

[コンポーネントのインストール]ダイアログで，[既存のパッケージへ追加｜新規パッケージに追加]の何れかの方法を選択後，[ユニットファイル名]に，Cool_Reg.PASを指定します。（フォルダーのパス付き）

[OK]ボタンをクリックすれば，インストールは完了です。パレットに，Nakanonが追加され，MainCoolとPopupCool及びCoolMenuBarの３つのコンポーネントが登録されます。

詳しくは，Delphiユーザーズガイドの102ページ，192ページ等を参照してください。

MenuImage Editorコンポーネントエディター

CoolMenuコンポーネントでは，メニューアイテムに表示するアイコンを指定するための便利なコンポーネントエディターが用意されています。

MainCoolコンポーネント・PopupCoolコンポーネントをダブルクリックするか，マウスの右クリックで表示されるメニュー項目　[ メニュー項目イメージの設定... ]　をクリックすると，以下のようなコンポーネントエディターのダイアログが表示されます。

PopupMenuがうまく表示されないときは．．．．

PopupCoolを使ったとき，実行時にポップアップメニューがうまく表示されないことがあります。（TTimerコンポーネントを一緒に使っているようなとき）そういった場合は，TPopupCoolコンポーネントを選択し，マウスの右クリックで表示されるメニュー項目　[ コンポーネントインデックスの変更]　をクリックすると，TPopupMenuコンポーネントとTPopupCoolコンポーネントの親フォーム上での，コンポーネントインデックスを可能な限り「０」に近くして，TTimerコンポーネントよりも早く生成されるように設定します。こうすることによって，ポップアップメニューの表示が可能になります。

TMenuItemProperty    プロパティエディター
TPopupCoolProperty  プロパティエディター
TMainCoolProperty    プロパティエディター

オブジェクトインスペクター上で，MainCool・PopupCoolのプロパティの設定をすることを支援するためのプロパティエディターです。

TCoolMenuBarEditor コンポーネントエディター

CoolMenuBarコンポーネント上で，マウスを右クリックすると表示されるメニューの，一番上の項目「メニュー項目の再描画」を実行すると，現在のMainMenuのメニューバー項目の変更をCoolMenuBarに反映させることができます。
