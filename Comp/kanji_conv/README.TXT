●機能説明

  JIS,EUCなどの文字コードの相互変換とBase64などの符号化エンコード／デコード
  機能をまとめたコンポーネントです。
  メールの送受信などインターネットアプリケーションに利用できます。
  すべてDelphiで扱いやすいString型で処理できます。
  
    JIS <-> ShiftJIS（半角カタカナ変換機能あり）
    EUC <-> ShiftJIS（半角カタカナ変換機能あり）
    Base64 Encode/Decode
    MIME Header Encode/Decode
    WEB code Encode/Decode

--------------------------------------------------------------------------
●使用方法

  ※C++Builderにインストールする場合の説明はありませんが、ソースを添付
    していますのでマニュアルに従ってインストールして下さい。

  コンポーネントをインストールするのに必要なファイルは、
    PzConv.dcu   // 本体
    PzConv.dcr   // パレットビットマップ

  環境設定のライブラリ検索パスの通ったディレクトリにファイルをコピーします。
     ....Delphi4\Lib   あるいは新規に ....Delphi4\UserLib など
  （\Libにはユーザ定義用パッケージdclusr40.dpkが用意されています）

  Delphiのマニュアルに従ってパッケージにコンポーネントをインストールします。
  コンポーネントパレット「Samples」に「PzConv」が登録されます。
  
  サンプルプログラムがディレクトリ「Sample」にありますので見てください。 

  コンポーネントのリファレンスはディレクトリ「Docs」にHTMLで記述しています。
    TPzConv.htm
