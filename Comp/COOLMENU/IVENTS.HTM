property OnDrawItem

TDrawMenuItemEvent = procedure(Menu: TMenu; Item: TMenuItem; Rect: TRect; State: TOwnerDrawState) of object;

コンポーネントによるデフォルトの描画を使わないで，アプリケーション開発者が自由にメニューアイテムに描画することを提供するイベントです。

例

procedure TForm1.PopupCool1DrawItem(Menu: TMenu; Item: TMenuItem; Rect: TRect; State: TOwnerDrawState);
begin
  PopupCool1.Canvas.FillRect(Rect);
  PopupCool1.Canvas.TextOut(Rect.Left + 10, Rect.Top, Item.Caption);
end;

property OnMeasureItem

TMeasureMenuItemEvent = procedure(Menu: TMenu; Item: TMenuItem; var Height: Integer) of object;

ItemHeightプロパティの設定に依らないで，メニューアイテムの高さを設定するためのイベントです。新しく設定したい高さは，Heightに指定して返します。

個別のメニューアイテムの高さを設定することもできます。

例

procedure TForm1.PopupCool1MeasureItem(Menu: TMenu; Item: TMenuItem; var Height: Integer);
begin
  if Item.MenuIndex = 3 then
  Height := 100;
end;

property OnMenuSelect

TMenuSelectEvent = procedure(Item: TMenuItem; Hint: String) of object;

メニューアイテムが選択状態になったことを通知するイベントです。Itemは選択状態のメニューアイテムで，Hintはそのメニューアイテムに設定した，ヒント文字列です。

例

procedure TForm1.MainCool1MenuSelect(Item: TMenuItem; Hint: String);
begin
  StatusBar1.SimpleText := Item.Caption + ' : ' + Hint;
end;
